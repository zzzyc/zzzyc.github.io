$A.$

题意：$n$件物品中有$m$件优等品，共$k$个人，每个人可以拿到$\frac{n}{k}$件物品，保证$k|n$，$Bob$会拿尽可能多的优等品数$x$，求其余$n-1$个人中可获得的最多的优等品数$y$，输出$x-y$。

数据范围：$1\leq t \leq 500,\ 2\leq n\leq 50, \ 0\leq m \leq n, 2 \leq k \leq n$

题解：要么$m\leq(k|n)$，那么就是$0$

否则就是$(k|n) - ((m-ev) / (k-1) + ((m - ev) % (k - 1) != 0)$

$B.$

题意： 给定一个只由$’‘$和$’.’$构成$n\times m$的矩阵，要求将$’.’$全变成$’‘$，每次要么修改一个元素，要么只能修改一行内的两个连续的$’.’$，只修改一个元素花费为$x$，修改连续两个元素花费为$y$，求花费最小值。

数据范围：$1\leq t \leq 500, \ 1\leq n \leq 100, 1\leq m \leq 1000,1\leq x,y \leq 1000$

题解：如果$x \times 2 \leq y$，那么置$y = 2 \times x$，分别枚举每行，每统计完一个连续的$’.’$序列，就计算下：能用$y$尽量用$y$，否则用$x$。

$C.$

题意： 往水壶中加水，热冷热冷这样的循环加水，每次加的水量一样。给定热水温度$h$和冷水温度$c$，求最接近给定温度$t$需要加多少次水。

数据范围：$1\leq c \leq h \leq 10^6, \ c \leq t \leq h$

题解：

(1)当$h==t$，为$1$

(2)当$h+c\geq 2 \times t$ 为$2$

(3)当$h + c \leq 2 \times t$

​ 此时答案不会是偶数，因为偶数的温度即$\frac{h+c}{2}$，这必然是最高温度，所以后面就可以无限逼近最高温度但是不到达。那么对于选取奇数次，必然是$k+1$次热，$k$次冷，此时可列式：$\frac{(h+c)\times k+h}{2\times k+1}=t$

解得：$k=\frac{h-t}{2\times t - h - c}$，为了防止边界判断一下再其左右即可。

$E.$

题意： 给定$n,k$，求一个元素不互相同且在$[1,n]$间的序列$a$，使得对每个非负数$x$，$x$对这个序列按照任意排列取模的结果都一样，输出序列$a$的数量，对$998544353$取模。

数据范围：$1\leq n,k \leq5\times 10^5$

题解： 手模下可以发现当且仅当序列中其他数都是序列中最小数的倍数时，才能保证$x$就是直接模最小数。

预处理下阶乘和逆元，然后直接$O(1)$求组合数，可以用$n / i$判断$i$为最小值时其倍数的个数
